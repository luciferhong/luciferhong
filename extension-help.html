루시퍼홍 부동산 도구모음

네이버부동산·아실·지인·월부에서 쓰던 유저스크립트들을 하나의 크롬 확장으로 모았습니다. 사이드 패널에서 원하는 기능만 켜고 끌 수 있습니다.

크롬 웹스토어에서 설치하기

설치 방법

  1. 위 설치하기 버튼을 눌러 웹스토어 페이지로 이동합니다.
  2. Chrome에 추가 → 확장 프로그램 추가를 클릭합니다.
  3. 브라우저 우측 상단 퍼즐(🧩) 아이콘에서 루시퍼홍 부동산 도구모음을 고정해 두면 편합니다.
브라우저 설정은 따로 건드릴 필요가 없습니다. 개발자 모드나 "파일 URL 액세스 허용" 같은 옵션을 켜지 않아도 바로 동작합니다. 다만 Chrome·Edge·웨일 등 크로미엄 계열 114 버전 이상이 필요합니다(요즘 브라우저면 자동 충족). 모바일 브라우저는 확장 설치를 지원하지 않습니다.

사용 방법

  1. 브라우저 우측 상단의 확장 아이콘(🏢)을 클릭하면 사이드 패널이 열립니다.
  2. 패널에서 사용할 기능의 체크박스를 켜거나 끕니다.
  3. 변경 후에는 해당 사이트를 새로고침해야 반영됩니다(패널의 "현재 탭 새로고침" 버튼 이용).

일부 기능은 처음에 꺼진 상태로 제공됩니다(아실 지도 실거래수 표시, 학군&환경&단지 표시). 필요할 때 직접 켜세요.

기능 목록

기능 제목을 누르면 설명이 열립니다. 괄호 안은 동작하는 사이트입니다.

네이버부동산 (new.land.naver.com 단지 페이지)

단지라벨 색상변경 네이버부동산
지도의 단지 라벨 색상을 바꿔 원하는 단지를 눈에 띄게 표시합니다.
부동산 매물 가격 필터 (based on 모느나님) 네이버부동산
매물 목록에 면적·가격 정보를 정리해 표시하고, 원하는 조건으로 매물을 필터링합니다. 값 복사 기능 포함.
네부 부동산 새창으로 열기 네이버부동산
중개소 이름을 클릭하면 해당 중개소 매물을 새 탭으로 엽니다.
네이버 부동산 매물 다운로드 네이버부동산
현재 단지의 매물 목록을 엑셀(.xlsx) 파일로 저장합니다.
네이버 부동산 중개소 다운로드 네이버부동산
단지에 등록된 중개소 정보를 엑셀로 저장합니다.

아실 (asil.kr)

NAC (New Asil Chart) 아실
아실 아파트 차트를 개선해 매매·전세 등 정보를 더 보기 좋게 표시합니다.
아실 여러단지비교 편하게 써보자 아실
여러 단지 비교 화면에서 지역·단지 선택과 패널 조작을 편하게 도와줍니다.
아실 차트 가격표 (실거래 바로가기·월세 제외 포함) 아실
차트에 가격표를 얹고, 실거래가로 바로 이동하며, 거래현황에서 월세를 제외해 보여줍니다.
아실차트 호갱노노처럼 아실
아실 차트를 호갱노노 스타일로 보이도록 조정합니다.
아실 지도에 실거래수 표시하기 기본 꺼짐
지도 위에 단지별 실거래 건수를 표시합니다. 기본적으로 꺼져 있으니 필요할 때 켜세요.
아실 학군&환경&단지 함께 표시 기본 꺼짐
지도에 초·중학교 학군, 백화점·마트·병원, 스타벅스 등을 함께 표시합니다. 필터 영역에 표시 버튼이 추가됩니다.
아실 중개소 연락처 다운로드 아실
지도에 표시된 중개소들의 연락처를 엑셀로 저장합니다.
아실 아파트 연식라벨 아실
지도 단지에 준공 연식 라벨을 표시하고, 연식·세대수 기준으로 구분해 봅니다.

수요·공급 표를 원하는 조건으로 필터링해 봅니다.

수요·입주 데이터를 더 상세하게 엑셀로 저장합니다.

자주 묻는 질문

기능을 켰는데 화면에 안 나타나요
사이드 패널에서 켠 뒤 해당 사이트를 새로고침했는지 확인하세요. 대부분 새로고침하면 적용됩니다.
설정이 다른 PC에서도 유지되나요
켜기/끄기 설정은 크롬 계정 동기화 저장소에 저장되어, 같은 계정으로 로그인한 다른 크롬에서도 유지됩니다.
개인정보를 수집하나요
아니요. 이 확장은 사용자 데이터를 수집하거나 외부로 전송하지 않습니다. 각 기능은 해당 사이트 화면을 보기 좋게 바꾸거나, 그 사이트의 데이터를 엑셀로 내보내는 용도로만 동작합니다.
