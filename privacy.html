개인정보 처리방침

서비스명: 홍부가 기가막혀 (luciferhong.github.io)
시행일: 2026년 4월 22일 · 최종 수정일: 2026년 4월 22일

본 처리방침은 홍부가 기가막혀(이하 "서비스")가 제공하는 부동산 분석 도구(hongbu.html, hongbuCloud.html, sise.html, population.html 등 GitHub Pages에 게시된 모든 페이지)의 개인정보 처리 기준을 설명합니다. 서비스는 운영자가 개인적으로 유지하는 정적 웹 페이지이며, 자체 서버나 데이터베이스를 운영하지 않습니다.

요약: 서비스는 사용자의 개인정보를 수집하거나 외부 서버로 전송하지 않습니다. 모든 사용자 데이터(메모, 사진, 설정)는 사용자 본인의 브라우저(IndexedDB/localStorage) 또는 사용자 본인의 Google Drive에만 저장되며, 운영자는 이에 접근할 수 없습니다.

1. 수집하는 개인정보 항목

서비스는 다음 데이터를 사용자 본인의 기기 또는 사용자 본인의 Google 계정에서만 처리합니다.

항목저장 위치용도
지도 위치 메모, 단지 메모 브라우저 IndexedDB (hongbu, hongbuMemo) 임장 기록 보관
사진 메모 (사용자가 직접 첨부한 이미지) 브라우저 IndexedDB 또는 Google Drive (HongbuMemo 폴더) 현장 사진 보관
지도 중심 위치, 줌 레벨, 필터 설정 브라우저 localStorage 재방문 시 이전 상태 복원
Google OAuth 액세스 토큰 (hongbuCloud.html) 브라우저 localStorage (만료 시간 포함) Google Drive 자동 재로그인

서비스는 이름, 이메일, 전화번호, 위치정보, 쿠키 기반 식별자 등 어떠한 개인식별 정보도 수집·저장·전송하지 않습니다.

2. Google API 사용 및 제3자 데이터 처리

2.1 Google Drive 연동 (hongbuCloud.html 전용)

hongbuCloud.html은 사용자가 자신의 Google Drive에 메모와 사진을 백업할 수 있도록 Google Drive API를 사용합니다. 이때 요청하는 OAuth 범위(scope)는 다음 한 가지입니다.

  • https://www.googleapis.com/auth/drive.file — 민감하지 않은(non-sensitive) 범위. 본 서비스가 생성한 파일에만 접근하며, 사용자의 기존 Drive 파일 전체를 읽거나 수정할 수 없습니다.

서비스는 사용자의 Google 계정 정보, 이메일, 프로필 이름을 요청하거나 저장하지 않습니다. 저장되는 파일은 사용자 Drive의 HongbuMemo 폴더(및 하위 폴더)에만 생성되며, 운영자는 사용자의 Drive에 접근할 수 없습니다.

본 서비스의 Google 사용자 데이터 사용은 Google API 서비스 사용자 데이터 정책을 준수하며, 이에는 "제한된 사용(Limited Use)" 요건이 포함됩니다. 구체적으로:

  • Google 사용자 데이터는 본 서비스의 메모·사진 저장 기능 제공 외의 목적으로 사용하지 않습니다.
  • Google 사용자 데이터를 광고, 분석, 프로파일링 목적으로 사용하거나 제3자에게 전송하지 않습니다.
  • 사용자의 명시적 동의 없이 Google 사용자 데이터를 사람이 열람하지 않습니다.

2.2 네이버 지도 API

지도 표시를 위해 NAVER Cloud Platform의 지도 API를 사용합니다. 지도 타일 요청에는 사용자의 IP 주소와 브라우저 User-Agent가 네이버 측에 전달될 수 있으며, 이는 네이버 클라우드 개인정보 처리방침을 따릅니다.

2.3 기타 제3자 리소스

서비스는 다음 CDN에서 라이브러리를 로드합니다. 각 CDN은 해당 운영사의 정책을 따릅니다.

  • Google Fonts, Google Identity Services (accounts.google.com, apis.google.com)
  • jsDelivr, unpkg (오픈소스 라이브러리 배포)

3. 개인정보 보유 및 파기

  • 브라우저 저장 데이터: 사용자가 브라우저 데이터 삭제, 시크릿 모드 종료, 또는 서비스 내 삭제 기능을 사용할 때까지 보관됩니다. 운영자는 이 데이터에 접근할 수 없습니다.
  • Google Drive 저장 데이터: 사용자 본인의 Drive 저장 정책을 따릅니다. 사용자가 Drive에서 파일을 삭제하거나 Google 계정 설정에서 본 앱의 액세스를 해제하면 즉시 연동이 중단됩니다.
  • OAuth 액세스 토큰: 발급 후 1시간 내 자동 만료되며, 로그아웃 버튼 클릭 시 즉시 폐기됩니다.

4. 사용자 권리

사용자는 언제든지 다음 작업을 직접 수행할 수 있습니다.

  • 데이터 열람/수정/삭제: 각 페이지 내 메모 관리 UI를 통해 수행.
  • Google Drive 액세스 해제: Google 계정 권한 관리 페이지에서 "홍부가 기가막혀" 앱을 제거.
  • 브라우저 로컬 데이터 전체 삭제: 브라우저 설정 → 사이트 데이터 삭제(luciferhong.github.io).

5. 보안

  • 서비스는 HTTPS(TLS)로만 제공됩니다 (GitHub Pages).
  • Google OAuth는 업계 표준 OAuth 2.0 + PKCE 흐름을 사용합니다.
  • 서비스가 자체 서버를 운영하지 않으므로 서버 측 침해 경로가 존재하지 않습니다.

6. 아동의 개인정보

본 서비스는 만 14세 미만 아동을 대상으로 하지 않으며, 아동으로부터 개인정보를 의도적으로 수집하지 않습니다.

7. 처리방침 변경

본 처리방침이 변경될 경우 본 페이지의 "최종 수정일"이 갱신되며, 중대한 변경 사항이 있을 경우 서비스 내 공지를 통해 사전 안내합니다.
