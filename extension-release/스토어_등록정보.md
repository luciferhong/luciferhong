# Chrome 웹스토어 미등록(Unlisted) 게시 준비물

제출 파일: `extension-release/luciferhong-tools-v1.4.0.zip` (manifest.json이 zip 루트에 위치)

## 1. 절차 체크리스트

1. [Chrome 개발자 대시보드](https://chrome.google.com/webstore/devconsole) 접속 → 개발자 계정 등록 (1회 $5, 결제 카드 필요)
2. **새 항목** → zip 업로드
3. 아래 등록정보 입력 → **공개 상태: 미등록(Unlisted)** 선택 → 심사 제출
4. 심사는 통상 수 시간~수일. 승인되면 스토어 링크가 활성화되고, 링크를 아는 사람만 설치 가능
5. 이후 업데이트: manifest `version` 올리고 zip 재빌드 → 대시보드에서 새 패키지 업로드

참고: "비공개(Private)"는 지정한 테스터 계정만 설치 가능한 별도 옵션입니다. 링크 공유 방식이면 **미등록(Unlisted)**이 맞습니다.

## 2. 스토어 등록정보 (복사용)

**이름**: 루시퍼홍 부동산 도구모음

**요약** (132자 이내):
네이버부동산·아실·지인·월부에서 쓰는 개인용 도구 모음. 단지라벨 색상변경, 매물 가격 필터, 차트 가격표, 시세·매물·중개소 다운로드 등을 사이드 패널에서 켜고 끌 수 있습니다.

**상세 설명**:
```
기존에 Tampermonkey 유저스크립트로 배포하던 부동산 분석 도구들을 하나의 확장으로 통합했습니다.

포함 기능 (18종, 사이드 패널에서 개별 on/off):
[네이버부동산] 단지라벨 색상변경, 매물 가격 필터, 매물/중개소 엑셀 다운로드, 매물 새창으로 열기
[아실] NAC(New Asil Chart), 차트 가격표, 호갱노노처럼, 여러단지비교, 거래현황 월세 제외, 실거래 바로가기, 지도 실거래수 표시, 학군&환경&단지 표시
[아파트지인] 수요-공급 표 필터링, 수요-입주 다운로드(지역/전국)
[월부] 투자공부인증 게시글 숨기기
[네이버카페] 튜터링데이 인증샷 Helper

사용법: 확장 아이콘 클릭 → 사이드 패널에서 원하는 기능 체크 → 해당 사이트 새로고침.
```

**카테고리**: 도구(Tools) 또는 생산성(Productivity)
**언어**: 한국어

## 3. 그래픽 자산

| 항목 | 규격 | 준비 상태 |
|---|---|---|
| 스토어 아이콘 | 128×128 PNG | `extension/icons/icon128.png` 그대로 업로드 |
| 스크린샷 (필수 1장 이상, 최대 5장) | 1280×800 또는 640×400, PNG/JPG | **직접 캡처 필요** — 추천: ① 사이드 패널 토글 화면 ② 네이버부동산 가격 필터 동작 화면 ③ 아실 차트 가격표 화면 |
| 프로모션 타일 (선택) | 440×280 | 미등록 게시에는 생략 가능 |

스크린샷 팁: 1280×800 크기 창에서 Win+Shift+S로 캡처 후 그림판에서 캔버스를 1280×800으로 맞추면 됩니다.

## 4. 개인정보 보호 탭 (심사 필수 항목)

**단일 목적(Single purpose) 설명**:
```
부동산 정보 사이트(네이버부동산, 아실, 아파트지인 등)의 화면 표시를 개선하고 표시된 데이터를 엑셀로 저장하는 개인용 도구 모음입니다.
```

**권한 사용 사유(Permission justification)**:
- `storage`: 각 기능의 켜기/끄기 설정을 저장하는 데만 사용합니다.
- `sidePanel`: 기능 on/off 토글 UI를 사이드 패널로 표시합니다.
- 호스트 접근(콘텐츠 스크립트 — new.land.naver.com, asil.kr, aptgin.com, weolbu.com, cafe.naver.com): 해당 사이트 화면에 기능(라벨 색상, 필터, 다운로드 버튼 등)을 추가하기 위해 필요합니다.

**데이터 수집 설문**: 모든 항목 "수집하지 않음" 선택.
(이 확장은 어떤 사용자 데이터도 수집·전송하지 않습니다. 설정은 브라우저 동기화 저장소에만 보관됩니다.)

**원격 코드(Remote code) 문항**: "아니요, 원격 코드를 사용하지 않습니다" 선택.
(v1.4.0에서 jQuery/clipboard.js/SheetJS/CryptoJS를 전부 확장에 번들했고, CDN 로드 코드는 번들 존재 시 실행되지 않는 폴백으로만 남아 있습니다.)

**개인정보처리방침 URL**: 데이터를 수집하지 않으므로 필수 아님. 요구될 경우 `https://luciferhong.github.io/privacy.html` 사용 가능.
